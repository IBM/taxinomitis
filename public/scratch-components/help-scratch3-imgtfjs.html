Machine learning models can be trained and stored ("hosted") on your own computer or on a shared server ("in the cloud").
When you change this, you need to reload your Scratch project. This lets the Scratch blocks for your project work correctly based on where your ML model is.
1. (optional) Save your project using "File" -> "Save to your computer"
2. Reload the browser page
3. (optional) Restore your project using "File" -> "Load from your computer"
