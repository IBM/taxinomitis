This block lets you start a microphone listening for sounds that a machine learning model can recognize. However, when you tried to use it, it didn't work because you don't seem to have a machine learning model.
The most likely reason is that you haven't trained a machine learning model. With sound projects, you need to train a model every time you open the Scratch page.
Or it could be that you started a model training, but didn't wait long enough for it to finish training.
