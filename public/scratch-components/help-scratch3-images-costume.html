This is how to use the block for a sprite:
1. Click on the blue-green Images icon in the toolbox.
2. Drag one of the image blocks into your recognise image block (either costume image or backdrop image)
If you use costume image your machine learning model will try to recognise the current costume for this sprite (ignoring any backdrop, any other sprites, or whether or not this sprite is hidden or off-screen)
If you use backdrop image your machine learning model will try to recognise what it can see at your sprite's location - including other sprites that overlap and any backdrop.
